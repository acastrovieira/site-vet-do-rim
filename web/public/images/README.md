# Fotos do Site — Vet do Rim

Coloque suas fotos nesta pasta. O site já está preparado para exibi-las.

## Arquivos esperados

| Arquivo | Onde aparece | Tamanho ideal |
|---|---|---|
| `dr-aline-hero.jpg` | Hero da home (ao lado do título) | 800×900px, vertical |
| `dr-aline-perfil.jpg` | Seção "Sobre" (foto principal) | 600×700px, vertical |
| `dr-aline-clinica.jpg` | Seção "Sobre" (foto secundária, clínica) | 800×600px, horizontal |

## Dicas para as fotos
- **Fundo neutro ou da clínica** — evitar fundo branco simples
- **Roupa profissional** — jaleco ou roupa clínica
- **Boa iluminação** — preferência por luz natural ou setup profissional
- **Formato JPG ou WebP** — qualidade alta (acima de 80%)

Após adicionar as fotos, avise o Antigravity para ativar as imagens no código.
